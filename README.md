# UmrahConnect 2.0 - Islamic Pilgrimage Marketplace

## 🕌 Project Vision
The world's first comprehensive, verified marketplace for all Islamic pilgrimage services - the "Amazon of Hajj & Umrah" with integrated vendor management, customer experience, and complete service ecosystem.

## 📊 Market Opportunity
- **Market Size:** $18+ billion global Islamic pilgrimage market
- **Target Markets:** 
  - Primary: India, Pakistan, Bangladesh, Indonesia (60%)
  - Secondary: Middle East (30%)
  - Tertiary: Western markets (10%)

## 💰 Revenue Projections
- **Year 1:** ₹3-5 Crores | 50,000+ customers, 500+ vendors
- **Year 3:** ₹25+ Crores | 500,000+ customers, 3,000+ vendors
- **5-Year:** ₹100+ Crores

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- PostgreSQL 14+
- MongoDB 6+
- Redis 7+
- Docker & Docker Compose

### Installation

```bash
# Clone repository
git clone https://github.com/IamTamheedNazir/umrahconnect-2.0.git
cd umrahconnect-2.0

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Run database migrations
npm run migrate

# Start development server
npm run dev
```
